# AAU BAM-1022 PM2.5 Data Preparation for OpenAQ

This repository contains the **data processing workflow** used to prepare the Addis Ababa University (AAU) GEOHealth BAM-1022 PM2.5 dataset for **OpenAQ submission**.  

The workflow ensures that each measurement is non-imputed, follows OpenAQ framework requirements, and is reproducible for future submissions.

---

## 📍 Station Information

- **Name:** AddisTASHGEOHealth  
- **Location:** Tikur Anbessa Specialized Hospital (TASH), Addis Ababa  
- **Coordinates:** 9.01523, 38.74291  
- **Instrument:** BAM 1022  
- **Parameter:** PM2.5  
- **Duration:** 3600 seconds (hourly measurements)  

This metadata applies to all measurements in the dataset and is included in every row during data preparation.

---

## 🔐 Data Policy

- Raw measurement data is **confidential** and **not included** in this repository.  
- Only processing scripts, sample outputs, and documentation are available publicly.  

This repository contains:  
- **R formatting script** – Converts raw BAM CSVs into OpenAQ-ready format  
- **Validation logic** – Ensures uniqueness and ISO-8601 datetime compliance  
- **Framework-compliant structure** – Metadata added, non-imputed values filtered  
- **Sample output** – Synthetic/demo data for reference and testing

## 📦 Output Format (OpenAQ Framework)

Each row represents **one non-imputed measurement** and includes the following columns:

| Column        | Description |
|---------------|-------------|
| `name`        | Station name |
| `location_code` | Unique station identifier |
| `latitude`    | WGS84 latitude of the station |
| `longitude`   | WGS84 longitude of the station |
| `duration`    | Duration of the measurement in seconds (e.g., 3600 for hourly values) |
| `parameter`   | Name of the pollutant (e.g., pm25) |
| `unit`        | Unit of the measurement (e.g., ug/m3) |
| `sensor_code` | Instrument used (e.g., BAM 1022) |
| `value`       | Measured value |
| `datetime`    | ISO-8601 timestamp with timezone (e.g., `2023-12-01T22:30:38+03:00`) |

> **Uniqueness Rule:**  
> The combination of `datetime + parameter + value` must be unique for all rows.

## ⚙️ How to Use

1. Place your confidential raw BAM CSV file(s) in the `input/` folder.  
2. Run the R script(s) in `aau_bam_openaq_formatting.R` to generate OpenAQ-ready CSVs in `output/`.  
3. Use `sample_output` for testing, reference, or documentation.
